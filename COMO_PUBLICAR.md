# 🚀 Como Publicar o App "Meu Dinheiro"

Siga o passo a passo abaixo. Não precisa saber programar!

---

## ✅ O que você vai precisar
- Um computador com internet
- Uma conta gratuita no GitHub (github.com)
- Uma conta gratuita no Vercel (vercel.com)

Tempo estimado: **10 a 15 minutos**

---

## PASSO 1 — Criar conta no GitHub

1. Acesse: **https://github.com**
2. Clique em **"Sign up"** (Criar conta)
3. Preencha nome, e-mail e senha
4. Confirme o e-mail que eles enviam

---

## PASSO 2 — Criar um repositório (pasta) no GitHub

1. Após entrar no GitHub, clique no **"+"** no canto superior direito
2. Clique em **"New repository"**
3. No campo **"Repository name"**, escreva: `meu-dinheiro`
4. Deixe marcado como **"Public"**
5. Clique em **"Create repository"**

---

## PASSO 3 — Enviar os arquivos do app

Dentro do repositório recém-criado:

1. Clique em **"uploading an existing file"** (link no meio da tela)
2. Arraste os 3 arquivos para a área indicada:
   - `index.html`
   - `manifest.json`
   - `sw.js`
3. Clique em **"Commit changes"** (botão verde lá embaixo)

✅ Seus arquivos já estão no GitHub!

---

## PASSO 4 — Publicar no Vercel (cria o link do app)

1. Acesse: **https://vercel.com**
2. Clique em **"Sign Up"** → escolha **"Continue with GitHub"**
3. Autorize o Vercel a acessar sua conta GitHub
4. Na tela principal, clique em **"Add New Project"**
5. Encontre o repositório **"meu-dinheiro"** e clique em **"Import"**
6. Não precisa mudar nada — clique direto em **"Deploy"**
7. Aguarde ~1 minuto enquanto publica

✅ Pronto! O Vercel vai te dar um link assim:
**https://meu-dinheiro.vercel.app**

---

## PASSO 5 — Compartilhar com a equipe

Mande o link pelo WhatsApp para cada pessoa.

Cada um instala no celular assim:

### iPhone (Safari):
1. Abra o link no Safari
2. Toque no botão de **Compartilhar** (quadrado com setinha)
3. Role a lista e toque em **"Adicionar à Tela de Início"**
4. Toque em **"Adicionar"**

### Android (Chrome):
1. Abra o link no Chrome
2. Toque nos **3 pontinhos** (canto superior direito)
3. Toque em **"Adicionar à tela inicial"**
4. Confirme tocando em **"Adicionar"**

---

## 📌 Informações importantes

- **É gratuito** para até 100 GB de tráfego por mês (mais do que suficiente)
- **Os dados de cada pessoa ficam salvos no próprio celular** — privados
- **Funciona offline** após o primeiro acesso
- Se precisar atualizar o app, basta substituir o `index.html` no GitHub e o Vercel atualiza automaticamente em ~1 minuto
